# 📝 Registration Form (DOM) Project # 1

A simple and responsive **Registration Form** built using **HTML5, CSS3, and JavaScript (DOM)**. This project demonstrates form design, user input handling, and basic DOM manipulation for learning JavaScript fundamentals.
